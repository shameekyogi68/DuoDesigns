# Glossary of Terms

| Term | Definition |
|------|------------|
| **Monorepo** | A single git repository containing multiple projects (Apps + Backend). |
| **JWT** | JSON Web Token used for secure authentication. |
| **Taxable Amount** | The part of the order total that is subject to GST (Subtotal + Shipping - Discounts). |
| **Variant** | A specific color/size combination of a product. |
| **Webhook** | An HTTP callback that allows Razorpay to notify the backend about events. |
| **Zustand** | The state management library used in the frontends. |
| **Vite** | The build tool/dev server used for frontends. |
| **Render** | The hosting platform for the Backend API. |
| **Cloudinary** | The cloud service used for image storage and transformation. |
