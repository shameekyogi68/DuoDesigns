# ✍️ Style Guide (UI & Copy)

Duo Designs is a bold, modern brand. Our UI and copy should reflect strength, simplicity, and professionalism.

## 1. Writing Tone
- **Direct & Bold:** Use active verbs. (e.g. "Track Your Order" instead of "Your order can be tracked here").
- **Minimalist:** Remove unnecessary words. (e.g. "Payment Secure" instead of "Your payment is processed securely").
- **Confident:** Avoid "maybe" or "if". (e.g. "We deliver to you" instead of "We might deliver to your area").

---

## 2. Text Conventions
- **Case:** 
  - USE UPPERCASE for all headings, menu items, and buttons (Bebas Neue).
  - Use Sentence case for long descriptions and body text (Barlow).
- **Numbers:** Always use digits (`5 items`, not `five items`).
- **Currency:** Use the **₹** symbol (e.g., `₹499.00`). Total amounts should always show two decimal places.

---

## 3. Interaction Labels (Standard)

| Feature | Standard Label |
| :--- | :--- |
| **Call to Action** | SHOP NOW / PAY NOW / ADD TO CART |
| **Error Title** | OOPS! / SOMETHING WENT WRONG |
| **Success Title** | ORDER PLACED! / LOGIN SUCCESSFUL |
| **Empty State** | NOTHING HERE YET |
| **Cart Action** | GO TO CHECKOUT |

---

## 4. Iconography Rules
- Use thin-line (1.5px stroke) icons to match the border width.
- Icons should always be **Solid Black** or **Pure White**.
- Never use multi-colored emoji icons in the UI. Use SVG icons for a premium look.

---

## 5. UI Layout Principles
- **Grid Alignment:** Everything must align to the left or center. Avoid complex "floating" elements.
- **Negative Space:** Use generous white space to allow bold headings to stand out.
- **F-Pattern:** Place the most critical information (Price, CTA, Logo) along the top and left of the screen.

## ⚠️ The "No-No" List
- ❌ NO rounded corners.
- ❌ NO pastel colors.
- ❌ NO "handwriting" fonts.
- ❌ NO animations that last longer than 300ms.
- ❌ NO passive-aggressive copy (e.g. "No thanks, I don't want to save money").
