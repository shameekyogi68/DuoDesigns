# 🖨️ Print Specifications

To ensure the highest quality results for your custom streetwear, please follow these technical guidelines.

## 1. File Requirements
- **Formats:** PNG (with transparency) preferred. JPG accepted (for full-canvas prints).
- **Resolution:** Minimum **300 DPI** recommended. Files below **150 DPI** will trigger a low-resolution warning.
- **Color Mode:** **RGB** (Standard for web uploads). Our rip software handles the CMYK conversion.

## 2. Print Areas
| Product Type | Max Dimensions | Location |
| :--- | :--- | :--- |
| **Oversized T-shirt** | 12" x 16" (A3) | Chest / Back |
| **Heavyweight Hoodie** | 10" x 12" | Lower Chest |
| **Trackpants** | 4" x 6" | Upper Thigh |
| **Mugs** | 8" x 3.5" | Full Wrap |

## 3. Safe Zones & Bleed
- Keep text and critical elements at least **0.5" away** from the canvas edges.
- Avoid ultra-thin lines (< 2pt) as they may not bond correctly with the fabric during the heat-press phase.

## 4. Transparency
- Ensure designs don't have stray white pixels around the edges.
- For apparel, "White" in the image will be printed unless transparency is handled correctly (DTF/DTG process).
