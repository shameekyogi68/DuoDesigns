# Product Catalogue

Duo Designs categorizes its offerings into five main pillars, as defined in `Product.model.js`.

| Category ID | Display Name | Features | Design Support |
|-------------|--------------|----------|----------------|
| `tshirt` | Regular Tees | S to XXL, Cotton | Upload / Premade |
| `oversized` | Oversized | Drop shoulder, Premium | Upload / Premade |
| `trackpants` | Trackpants | Comfort fit, S to XL | Premade only |
| `mug` | Mugs | 11oz, 15oz | Upload / Premade |
| `keychain` | Keychains | Acrylic, Double side | Upload / Premade |

## Inventory Rules
- Products are marked as **Out of Stock** automatically if every color/size variant reaches 0.
- **Critical Stock** alerts appear in the Admin Dashboard when stock drops below 10 units.
