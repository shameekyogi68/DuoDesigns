# Admin UI Components

The Admin Panel UI is built for efficiency and data visibility.

## 📊 Dashboard Modules
- **StatsCard**: Compact components showing single metrics (e.g., "Today's Sales").
- **RevenueChart**: Interactive line graph using `Recharts`.
- **TopProductsGrid**: Ranking table of items by sales volume.

## 📋 Data Display
- **DataTable**: A robust table with pagination, search, and CSV export.
- **Badge**: Colored indicators for order status (`pending`, `paid`, `dispatched`).

## ⚙️ Layout
- **Sidebar**: Collapsible navigation with active state highlighting.
- **Navbar**: Breadcrumbs and admin profile menu.
