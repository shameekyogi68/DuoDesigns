# Order Management Workflow

Managing the lifecycle of a Duo Designs order from "Paid" to "Delivered".

## 1. Monitoring
Browse the **Orders** page with its real-time filters. Focus on the `placed` (paid but not confirmed) status first.

## 2. Processing (Order Detail)
- **Confirm**: Click "Confirm Order" to move it into the preparation queue.
- **Design Retrieval**: If the item is custom, click the Design URL to download the high-res file for printing.
- **Invoice**: Download the Tax Invoice PDF for inclusion in the physical package.

## 3. Shipping (Dispatch)
When the package is ready:
1. Select "Dispatch".
2. Enter the **Courier Name** (e.g., Delhivery).
3. Enter the **Tracking Number**.
4. Submit. The system will automatically email the customer.
