# Admin Page Reference

The admin panel is organized by operational function.

| Page Name | Description |
|-----------|-------------|
| **Dashboard** | Revenue charts, pending task counts, and low stock items. |
| **Orders** | Filterable list of all orders with status management. |
| **Order Detail** | Full order breakdown, design download per item, and dispatch tool. |
| **Products** | Catalogue management (Add/Edit items). |
| **Stock** | Bulk stock update tool for variants. |
| **Customers** | Registered user list and order histories. |
| **Coupons** | Create/Delete promotional codes. |
| **Shipping** | Pincode management and bulk CSV import. |
| **Partners** | Sales attribution log for the agency. |
| **Payments** | Audit log for all Razorpay transactions. |
