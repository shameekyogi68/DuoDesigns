# UI Components & Icons

The core building blocks of our interface.

## 🖱️ Interactive Elements

### Buttons
- **Primary**: Solid `--black` with `--white` text. Electric lime glow on hover.
- **Accent**: Solid `--accent` with `--black` text. Used for "Buy Now".
- **Outline**: `--light-gray` border. Used for "Add to Wishlist".

### Cards
- **Product Card**: Subtle 1px border (`--light-gray`). No shadow by default.
- **Hover State**: Lifts 4px (`transform: translateY(-4px)`) with a soft `--shadow-md`.

## 🎨 Icons
We use **Lucide React** for all iconography across the 4 modules.

| Icon name | Context |
|-----------|---------|
| `ShoppingBag` | Cart navigation. |
| `User` | Profile and Login. |
| `Search` | Global search bar. |
| `Truck` | Order tracking. |
| `Heart` | Wishlist and Likes. |
| `CheckCircle`| Success states and validation. |
