# Color Palette

Our colors are defined as CSS variables to allow for easy theme switching and consistent branding.

## 🌑 Primary Palette

| Variable | Hex | Sample | Description |
|----------|-----|--------|-------------|
| `--black`| `#0A0A0A` | ◼️ | Main text, background elements. |
| `--white`| `#F8F8F4` | ◻️ | Page background, card text. |
| `--accent`| `#C8FF00`| 🟩 | Call-to-action, primary buttons. |

## 🩶 Secondary Palette

| Variable | Hex | Description |
|----------|-----|-------------|
| `--gray` | `#6B6B6B` | Secondary text, icons. |
| `--light-gray` | `#E5E5DF` | Borders, dividers, disabled states. |
| `--card-bg` | `#F2F2EC` | Product card and container backgrounds. |

## 🚨 Status Colors

| Variable | Hex | Use Case |
|----------|-----|----------|
| `--error` | `#EF4444` | Validation errors, Cancelled status. |
| `--success` | `#22C55E`| Order placed, Profile updated. |
| `--warning` | `#F59E0B`| Pending payment, Stock alert. |
