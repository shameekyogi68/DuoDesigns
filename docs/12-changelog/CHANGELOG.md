# 📜 Version Changelog

Tracking the evolution of the Duo Designs platform.

## [v1.0.0] - 2025-03-13
### ✨ Added
- **Core Engine:** Full Node.js Express API.
- **Frontend v1:** Complete customer storefront (11 pages) and Admin Dashboard.
- **Payments:** Secure Razorpay integration with Webhook support.
- **Taxes:** Dynamic GST engine for Intrastate/Interstate.
- **Customization:** Cloudinary-backed high-res design upload system.
- **Invoicing:** Dynamic PDF generation for all orders.
- **Security:** JWT Auth, Rate limiting, Admin 2FA, and Sentry error tracking.
- **Logistics:** Return/Replacement flow, Razorpay Refund API, and Shiprocket tracking integration.
- **Performance:** PWA support, OptimizedImage component, and Cloudinary auto-transformations.
- **Compliance:** Full legal policies, GST data export, and Print specifications.
- **Engagement:** Wishlist, Product Reviews, and WhatsApp notifications.
