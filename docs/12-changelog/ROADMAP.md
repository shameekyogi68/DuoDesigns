# 🗺️ Project Roadmap (2025-2026)

Future features and platform enhancements planned to scale Duo Designs.

## 🚀 Q2 2025: Advanced Customization
- **Design Studio 2.0:** Allow users to add text, change font colors, and move their uploaded graphics on the mockup.
- **Bulk Orders:** A dedicated B2B enquiry form for companies and college fests.
- **Wishlist:** Allow customers to save items for later.

## 🛠️ Q3 2025: Operational Efficiency
- **Shiprocket Integration:** Automate courier booking and real-time tracking updates from within the Admin Dashboard.
- **inventory Sync:** Integrations with warehouse software for multi-location inventory.
- **WhatsApp Alerts:** Send Order/Dispatch notifications via WhatsApp API.

## 📈 Q4 2025: Growth & SEO
- **Referral Program:** Customers can earn discounts by inviting friends.
- **Blog Section:** Street-wear culture and designer spotlights to drive organic traffic.
- **Apple Pay / GPay Direct:** One-click checkout options bypassing the standard Razorpay selection.

## 🌐 2026: Global Expansion
- **International Shipping:** Handling USD pricing and multi-currency checkouts.
- **Designer Marketplace:** Allow artists to upload their designs and earn royalties on every sale.
- **PWA / Mobile App:** Launching native iOS and Android apps for the storefront.
    - **Tech Stack:** React Native (sharing core React logic).
    - **Features:** Native push notifications (Firebase), biometric login, and deep-linking from WhatsApp.
