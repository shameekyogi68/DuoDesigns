# Frontend Developer Roles

Each frontend in our ecosystem serves a unique audience and requires a specific mindset.

## 👕 Customer App Specialist
- **Focus**: Mobile performance, SEO, and visual "WOW" factor.
- **Goal**: Conversion. 
- **Skill**: Mastery of CSS animations and React Suspense.

## 🛠️ Admin Panel Specialist
- **Focus**: Efficiency and data density.
- **Goal**: Operational speed.
- **Skill**: Mastery of complex tables, forms (React Hook Form), and dashboard layouts.

## 📈 Revenue Tracker Specialist
- **Focus**: Financial visualization and reporting.
- **Goal**: Transparency for partners.
- **Skill**: Mastery of `recharts` and client-side PDF generation.
