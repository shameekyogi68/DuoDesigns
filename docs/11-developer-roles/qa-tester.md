# QA & Product Roles

Ensuring the vision is delivered without bugs.

## 🧪 QA Tester
- **Responsibility**: Manual validation of the "Golden Path" (Product Selection -> Checkout -> Payment Success).
- **Tooling**: Browser DevTools (Network throttler), Postman, and real mobile devices (iPhone/Pixel).

## 📋 Product Manager
- **Responsibility**: Prioritizing the backlog and defining the roadmap based on sales data from the Admin dashboard.
- **Goal**: Identifying which product categories or features (e.g., "Mugs") are driving revenue and doubling down on them.
- **Documentation**: Keeping these docs updated as the system evolves.
