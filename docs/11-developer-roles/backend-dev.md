# Backend & DevOps Engineer

The backbone of the Duo Designs ecosystem.

## 🛡️ Backend Focus
- **Security**: Hardening the API against common attacks.
- **Third-Party Services**: Managing the deep integrations with Razorpay, Cloudinary, and Gmail.
- **Fulfillment Logic**: Maintaining the accuracy of GST calculations and invoice generation.

## 🚀 DevOps Focus
- **Availability**: Monitoring Render and MongoDB Atlas for uptime.
- **Secrets Management**: Ensuring production environment variables are never leaked.
- **Logs**: Managing Winston/Sentry logs for post-mortem analysis of failures.
