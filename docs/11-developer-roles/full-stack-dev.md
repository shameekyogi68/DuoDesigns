# Full Stack Developer Role

As a Full Stack Developer, you are responsible for the "Feature Vertical" — from the database schema to the UI implementation.

## 🛠️ Key Responsibilities
- **Data Integrity**: Designing Mongoose models that are performant and scalable.
- **API Performance**: Ensuring endpoints resolve in <200ms with optimized MongoDB queries.
- **UI Fidelity**: Implementing React components that match the Figma/Design tokens exactly.
- **State Flow**: Ensuring state is synced between logic (Zustand) and UI.

## 📁 Critical Files to Know
- `backend/src/models/`: Where the truth lives.
- `duo-designs-customer/src/store/`: Where the operational logic lives.
- `render.yaml`: Where the cross-app dependencies are defined.
