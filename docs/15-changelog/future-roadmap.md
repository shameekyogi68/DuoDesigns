# Future Roadmap

Planned high-priority features for Duo Designs.

## 🌙 Q3 2025: App Expansion
- **Native Android/iOS**: Using Capacitor to wrap the existing React customer app for push notifications.
- **Bulk Discounts**: Logic for B2B orders directly in the storefront.

## 🤖 Q4 2025: AI Integration
- **AI Design Assistant**: Using DALL-E/Stable Diffusion to help users generate custom prints from text prompts in the `CustomDesign` view.
- **Smart Sizing**: AR-based size recommendation tool.

## 🌍 Q1 2026: Global Launch
- **International Shipping**: Integration with FedEx/DHL.
- **Multi-Currency**: Automatic conversion based on user IP.
