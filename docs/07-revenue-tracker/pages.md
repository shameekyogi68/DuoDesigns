# Revenue Tracker Pages

The portal consists of five main operational views.

| Page Name | Description |
|-----------|-------------|
| **Dashboard** | Overview of lifetime earnings, current month sales, and growth charts. |
| **Sales Log** | Itemized list of every order attributed to the agency. |
| **Monthly Settlements** | Consolidated monthly view for billing and payout status. |
| **Reports** | Custom range filter to export earnings for specific periods. |
| **Settings** | Manage agency profile and notification preferences. |

## 🛡️ Security Note
The **Sales Log** masks customer sensitive data (e.g., full address or phone number) to ensure privacy while still allowing the agency to identify successful referrals.
