# 📧 Email Notification System

Duo Designs uses **Nodemailer** with a Gmail SMTP relay to send beautiful, branded transactional emails.

## 🛠️ Infrastructure
- **Provider:** Gmail (App Passwords).
- **Service:** `src/services/email.service.js`
- **Engine:** Dynamic HTML templates with string replacement.

---

## 📋 The 5 Core Templates

### 1. **OTP Verification**
- **Trigger:** `POST /api/auth/send-otp`.
- **Content:** "Your Duo Designs login code is 123456. Valid for 10 minutes."

### 2. **Order Confirmation (Tax Invoice)**
- **Trigger:** Payment verified successfully.
- **Content:** Order summary, branding, and an **attached PDF Invoice** generated by `invoice.service.js`.

### 3. **Order Dispatched**
- **Trigger:** Admin clicks "Dispatch" in the dashboard.
- **Content:** "Your Duo Designs gear is on the way!" + Tracking Link + Courier Name.

### 4. **Order Delivered**
- **Trigger:** Admin clicks "Deliver".
- **Content:** "Package received. Share your look with us on Instagram @duodesigns!"

### 5. **Account Critical**
- **Trigger:** Password reset (future) or Address change.

---

## 🧪 Implementation Snippet

```javascript
const transporter = nodemailer.createTransport({
    service: 'gmail',
    auth: {
        user: process.env.SMTP_EMAIL,
        pass: process.env.SMTP_PASSWORD
    }
});

exports.sendOrderConfirmation = async (userEmail, orderData, invoicePath) => {
    await transporter.sendMail({
        from: '"Duo Designs" <hello@duodesigns.in>',
        to: userEmail,
        subject: `Your Order #${orderData.orderNumber} is confirmed!`,
        html: generateOrderHTML(orderData),
        attachments: [{
            filename: `invoice-${orderData.orderNumber}.pdf`,
            path: invoicePath
        }]
    });
};
```

---

## ⚠️ Deliverability Rules
1. **No Spam:** Only transactional emails are allowed via this relay. No marketing newsletters.
2. **Branding:** All emails must use the Duo Designs logo and the **Lime Accent** footer.
3. **App Password:** Ensure `SMTP_PASSWORD` is an 16-character Google App Password, NOT the personal password.
