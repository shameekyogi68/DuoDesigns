# 🚀 Duo Designs

Premium street-wear custom-print e-commerce platform for India.

## 📖 Documentation
All project documentation, including architecture, development guides, and business rules, can be found in the `docs/` folder.

**[Start Here: Documentation Home](./docs/README.md)**

---

## 🏗️ Project Structure
- **/customer-app:** React + Vite storefront for customers.
- **/admin-app:** React + Vite dashboard for platform management.
- **/backend:** Node.js + Express API engine.

---

## 🛠️ Quick Setup
1. **Backend:**
   ```bash
   cd backend && npm install && npm run dev
   ```
2. **Frontend:**
   ```bash
   cd customer-app && npm install && npm run dev
   ```

---

## 🎨 Tech Stack
- **Frontend:** React, Vite, Vanilla CSS, Zustand.
- **Backend:** Node.js, Express, Mongoose, Razorpay, Cloudinary, pdf-lib.
- **Database:** MongoDB Atlas.

---

## 📜 Business Rules Recap
- **Prepaid Only:** No COD supported.
- **Indian Market:** Focused on domestic pincodes and GST taxation.
- **Custom Prints:** PDF Tax Invoices generated for all orders.
